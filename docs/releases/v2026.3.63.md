# v2026.3.63 — 2026-03-06

## Last Seen and Activity log

Reduces recorder and Activity (logbook) updates for the Last Seen and Last Seen Latency diagnostic sensors.

### Changed
- **Last Seen** and **Last Seen Latency** — State is now written only when the displayed value actually changes: at most once per minute for Last Seen (rounded to the minute) and at most once per 30 seconds for Last Seen Latency (rounded to 30s). The coordinator still updates at full telemetry rate; only these two entities skip writing when the rounded value is unchanged, so the Activity window and recorder stay quiet while graphs remain accurate.
