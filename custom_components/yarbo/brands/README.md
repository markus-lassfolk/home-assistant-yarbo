# Yarbo Brand Assets

This directory contains brand assets for the Yarbo Home Assistant integration.

## Files

- `icon.png` — 256x256 integration icon (dark background, white robot silhouette)
- `icon@2x.png` — 512x512 high-DPI icon
- `logo.png` — Full Yarbo logo (horizontal layout)
- `logo@2x.png` — High-DPI logo

## Submitting to home-assistant/brands

To submit brand assets to the official Home Assistant brands repository:

1. Fork [home-assistant/brands](https://github.com/home-assistant/brands)
2. Create directory: `custom_integrations/yarbo/`
3. Add icon files following the [brands contribution guide](https://github.com/home-assistant/brands/blob/master/CONTRIBUTING.md)
4. Open a Pull Request

### Requirements

- SVG format preferred, PNG fallback (256x256 minimum)
- Must follow Yarbo brand guidelines
- Icon must be a standalone icon (no text)
- Logo may include the Yarbo wordmark

### Icon Specifications

- Square format: 256x256px
- Background: Transparent or dark (#1c1c1e)
- Style: Flat/minimal
